# N Queens Problem
N - Queens problem is to place `n queens` in such a 
manner on an `n x n` chessboard that no queens attack 
each other by being in the same row, column or diagonal.

## How to run
```shell
$ python3 main.py <board-size>
```

## Example
```shell
$ python3 main.py 3
  No solution found

$ python3 main.py 4
  -----------------
  |   | 1 |   |   |
  -----------------
  |   |   |   | 1 |
  -----------------
  | 1 |   |   |   |
  -----------------
  |   |   | 1 |   |
  -----------------
```
